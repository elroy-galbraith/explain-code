# explain-code marketplace

A small Claude Code / Cowork plugin marketplace containing the **explain-code** skill.

`explain-code` turns a piece of code — a change (diff/branch/PR), a whole feature,
a module, or an unfamiliar codebase — into a single self-contained HTML page
designed to serve two readers at once:

- **The hurried PR reviewer** — every section leads with a 2–3 sentence summary,
  and a short *test-first gate* at the top lets a reviewer who already understands
  the code confirm it in seconds and skip the rest.
- **The new hire** — the full, beginner-friendly walkthrough is one click away
  behind each summary, with diagrams and code snippets, so onboarding onto a
  feature or codebase can go as deep as they need.

It also fixes two common annoyances: quiz answers are shuffled at render time
(they don't always land in the same position), and the boilerplate HTML/CSS/JS is
generated by a Python script rather than hand-written each run.

## Install

In an interactive Claude Code / Cowork session:

```
/plugin marketplace add elroy-galbraith/explain-code
/plugin install explain-code@explain-code-marketplace
```

(Replace `elroy-galbraith/explain-code` with wherever you host this repo.)

You can also install the packaged `.plugin` file directly in the Cowork desktop
app by dropping it into a chat.

## What's inside

```
.claude-plugin/marketplace.json     # marketplace manifest
explain-code/                       # the plugin
├── .claude-plugin/plugin.json      # plugin manifest
└── skills/explain-code/
    ├── SKILL.md                    # the skill instructions
    └── scripts/
        ├── render.py               # JSON spec -> HTML renderer
        └── sample_spec.json        # worked example
```

## Try the renderer directly

```bash
python3 explain-code/skills/explain-code/scripts/render.py \
  explain-code/skills/explain-code/scripts/sample_spec.json -o sample.html
```

Open `sample.html` in a browser. Only the Python 3 standard library is required.

just diffs), and adds a Python-rendered HTML/CSS/JS pipeline and shuffled quiz
answers on top.
