explain-code

Test-first code walkthroughs — pass the gate and skip the explanation.

explain-code turns a piece of code — a PR, a whole feature, or a module you're onboarding onto — into a single self-contained HTML page. A short diagnostic gate runs first: ace it and the deep walkthrough stays collapsed, you skip straight to the quiz; miss a question and the full explanation expands automatically. Every section leads with a short summary that's always visible, with a beginner-friendly deep dive one click away. One artifact serves both the 60-second PR review and the hour-long onboarding read.

Live demos

Each demo is rendered straight from a JSON spec by the same render.py this plugin ships — nothing hand-written, nothing that can drift. Try the gate: answer honestly and see the walkthrough adapt.

Install

Inside an interactive Claude Code or Cowork session:

/plugin marketplace add elroy-galbraith/explain-code
/plugin install explain-code@explain-code-marketplace

Then just ask Claude to explain a diff, a PR, a feature, or a module — the skill triggers on phrases like "explain this PR" or "walk me through this module."
